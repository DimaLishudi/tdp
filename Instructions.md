## Installation

```bash
$ npm install
```

## Running database container
```bash
$ docker compose up -d
```
### Shutdown

```bash
$ docker compose down
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
